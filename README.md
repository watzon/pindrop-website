# Pindrop Website

Marketing website for [Pindrop](https://github.com/watzon/pindrop) - A Mac-native AI dictation app.

Built with [Astro](https://astro.build) and deployed on [Vercel](https://vercel.com).

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Structure

- `src/pages/` - Page routes (index, blog, community)
- `src/components/` - Reusable Astro components
- `src/content/` - Blog posts (MDX)
- `src/styles/` - Global CSS and Tailwind config

## Deployment

Automatically deploys to Vercel on push to `main`.
